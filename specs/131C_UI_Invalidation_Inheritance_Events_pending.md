# UI Invalidation for Inheritance Events

## Description
Ensure frontend cache invalidation and view refreshes react to the new inheritance change events without manual reloads.

## Requirements
- Frontend event handling recognizes new inheritance change events
- Cache invalidation updates capability details and business domain views
- API client types include the new events where applicable
- Test mocks include the new events to prevent unhandled requests
- UI updates occur without manual refresh after reparenting operations
